🚀 Getting Started

BIG DADDY V3 supports up to 3 simultaneous WhatsApp bots per account. Each bot has its own slot, console and configuration.

  1. Go to Deploy Bot and choose a slot (1, 2 or 3)
  2. Enter your WhatsApp number with country code (no spaces, no +)
  3. Select a command prefix and click Start Deployment
  4. Open WhatsApp → Settings → Linked Devices → Link a Device
  5. Enter the pairing code — your bot will connect automatically

📦 Bot Slots

Each slot is completely independent:

  • Separate phone number and session
  • Individual prefix and console
  • Independent uptime monitoring

⌨️ Commands

[prefix]help — Show all commands [prefix]menu — Main menu [prefix]sticker — Create stickers [prefix]ai — AI assistant [prefix]play — Play audio [prefix]tts — Text to speech [prefix]uptime — Check uptime

Replace [prefix] with your chosen prefix (., /, !, ', #)
